Plug-In EV Charging Station Tracker

Provide an informative and visually appealing tool for users to find EV charging stations in Austin, TX

Project Overview

In this project, I will be exploring and visualizing the Electric Vehicle (EV) charging station dataset provided. The goal of this project is to create an interactive dashboard that allows users to search for EV charging stations in a particular zipcode. The dashboard will consist of several visualizations, including:

  • A map that displays the location of EV charging stations in the typed zipcode.
  • A heatmap that shows the number of charging stations by zipcode.
  • A packed bubbles that shows the EV Charging Station by facilities and zipcode.
  • [DataSource] The Official City of Austin Open Data Portal - https://data.austintexas.gov
