Project Overview

This project is to create an interactive Power BI dashboard for tracking emergency supply procurement in Ukraine. The dashboard provides real-time data visualization, including KPIs to monitor procurement efficiency, timelines, and resource allocation. Users can also view geographical maps displaying supply distribution across different regions, enabling quick identification of areas requiring immediate attention. The tool aims to enhance decision-making and streamline the management of emergency resources in crisis situations.
